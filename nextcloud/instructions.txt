Step 1: Edit fstab to mount the drive(s).
sudo blkid
mkdir /mnt/hd1
sudo vim /etc/fstab
UUID=<UUID> /mnt/hd1 ext4 defaults,nofail 0 0
systemctl daemon-reload
sudo mount -a

If neccesary, apply correct permissions to all files on drive
sudo chown -R www-data:www-data /mnt/hd1
sudo chmod -R 777 /mnt/hd1

Step 2: Start Nextcloud
AFTER the above steps, start up the nextcloud containers. If said containers are already running, restart them.
Navigate to dir containing nextcloud files (in my homelab repo)
docker compose up --build --pull always --wait

Access the aio interface at https://ip.address.of.server:8080/
